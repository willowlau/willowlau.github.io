CentOS7安装部署GitLab CE社区版并且修改默认Nginx

# CentOS7安装部署GitLab CE社区版并且修改默认Nginx

官方中文：https://www.gitlab.cc/downloads/#centos7 (opens new window)

官方英文：https://about.gitlab.com/downloads/#centos7 (opens new window)

# 安装依赖库

在 Centos 7 系统上, 下面的命令将在系统防火墙里面开放HTTP和SSH端口

$ sudo yum install curl policycoreutils openssh-server openssh-clients
$ sudo systemctl enable sshd
$ sudo systemctl start sshd
$ sudo yum install postfix
$ sudo systemctl enable postfix
$ sudo systemctl start postfix
$ sudo firewall-cmd --permanent --add-service=http
$ sudo systemctl reload firewalld

# 下载RPM安装包

国内用户推荐手动下载清华大学官镜像站的RPM安装包进行安装，访问：https://mirrors.tuna.tsinghua.edu.cn/gitlab-ce/ (opens new window) 选择系统版本使用下面的命令进行下载安装

$ curl -LJO https://mirrors.tuna.tsinghua.edu.cn/gitlab-ce/yum/el7/gitlab-ce-8.14.3-ce.0.el7.x86_64.rpm
$ rpm -i gitlab-ce-8.14.3-ce.0.el7.x86_64.rpm

# 启动与管理GitLab

$ sudo gitlab-ctl reconfigure
$ sudo gitlab-ctl status
$ sudo gitlab-ctl stop
$ sudo gitlab-ctl restart
$ sudo ps aux | grep runsvdir

# 切换Nginx

禁用自带Nignx服务器

$ sudo vi /etc/gitlab/gitlab.rb
1

设置nginx为false

nginx['enable'] = false
1

# 修改域名

external_url 'http://your.domain'
1

重启 nginx, 重启gitlab

$ sudo gitlab-ctl reconfigure
$ sudo service nginx restart
$ sudo gitlab-ctl start
